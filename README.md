# FPS: First Pc's

Projeto de Front-End desenvolvido para o Checkpoint 3 da FIAP, disciplina de Frontend, turma 1ESPI.

## Descricao

A FPS: First Pc's e uma loja gamer ficticia focada em PCs gamers, perifericos, upgrades e setups personalizados. O site apresenta uma Home com chamada principal, uma pagina de portifolio com produtos e promocoes, uma pagina Sobre com a historia da marca e uma pagina de Contato com formulario validado em JavaScript.

## Tecnologias utilizadas

- HTML5 semantico
- CSS3
- JavaScript puro
- Git e GitHub
- GitHub Pages

## Funcionalidades

- Navegacao entre Home, Portifolio, Sobre e Contato.
- Hero section com apresentacao da loja FPS: First Pc's.
- Cards de beneficios e produtos em destaque.
- Portifolio com cards de produtos gamers e filtro por categoria.
- Tabela de precos e promocoes.
- Pagina Sobre com historia, missao, visao, valores e diferenciais.
- Formulario de contato com validacao de campos obrigatorios e email.
- Footer dinamico com ano atualizado automaticamente.
- Layout responsivo para desktop, tablet e celular.

## Como executar localmente

1. Clone o repositorio:

```bash

```

2. Acesse a pasta do projeto:

```bash
cd CP3_front_end_0524
```

3. Abra o arquivo `index.html` no navegador.
