# FPS: First Pc's

Projeto de Front-End desenvolvido para o Checkpoint 3 da FIAP, disciplina de Frontend, turma 1ESPI.

## Descrição

A FPS: First Pc's é uma loja gamer fictícia focada em PCs gamers, periféricos, upgrades e setups personalizados. O site apresenta uma Home com chamada principal, uma página de portfólio com produtos e promoções, uma página Sobre com a história da marca e uma página de Contato com formulário validado em JavaScript.

## Tecnologias utilizadas

- HTML5 semântico
- CSS3
- JavaScript puro
- Git e GitHub
- GitHub Pages

## Funcionalidades

- Navegação entre Home, Portfólio, Sobre e Contato.
- Hero section com apresentacao da loja FPS: First Pc's.
- Cards de benefícios e produtos em destaque.
- Portfólio com cards de produtos gamers e filtro por categoria.
- Tabela de preços e promoções.
- Página Sobre com história, missão, visão, valores e diferenciais.
- Formulário de contato com validação de campos obrigatórios e e-mail.
- Footer dinâmico com ano atualizado automaticamente.
- Layout responsivo para desktop, tablet e celular.


2. Acesse a pasta do projeto:

```bash
cd CP3_front_end_0524
```

3. Abra o arquivo `index.html` no navegador.
